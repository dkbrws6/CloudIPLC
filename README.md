**线路质量方面**

这是CloudIPLC被夸得最多的地方。香港CMI线路三网回程质量稳定，实测下载速度基本能跑满100Mbps的理论峰值。移动用户直连体验最好，电信和联通虽然去程绕美国，但回程走CMI也不算太慢。有用户反馈说晚高峰时段偶尔会有波动，但总体来说比很多超售严重的商家要好得多。

沪日IPLC专线的延迟表现确实厉害，实测26毫秒左右，这个数据已经超过了大部分CN2 GIA线路。对于玩日服游戏或者需要低延迟连接的场景，这条线路的体验真的很香。而且因为是专线，所以晚高峰也不会受到太大影响，稳定性比较靠谱。
